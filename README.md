## 📋 Project Overview

**Royal Template** is a complete hotel and booking website template featuring room reservations, contact forms, and modern design. Built with Bootstrap 5, Font Awesome icons, and Swiper slider for a premium user experience.

## ✨ Features

- ✅ **Complete Booking System** - Room reservation functionality
- ✅ **Responsive Design** - Mobile-first approach
- ✅ **Multiple Pages** - Home, Booking, Contact pages
- ✅ **Modern UI/UX** - Clean and professional design
- ✅ **Image Slider** - Swiper.js integration
- ✅ **Contact Form** - Functional contact page
- ✅ **Font Awesome Icons** - 7.1.0 version included
- ✅ **Bootstrap 5.2.3** - Latest stable version
